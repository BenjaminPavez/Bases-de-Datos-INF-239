3) Una cancion puede ser agregada a favoritos siempre y cuando se haya reproducido alguna vez

4) Cuando se escucha, el numero de veces escuchadas solo se modifica en la tabla reproduccion

5) La fecha actual hay que escribirla manualmente, puesto que es la forma mas rapida para no esperar un dia para hacer las comprobaciones
